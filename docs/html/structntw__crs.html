Detailed Description

A Compressed Row Storage struct.

Used to save web graph tables in RAM.

The represented matrix can be easily converted into a probability matrix and the dangling nodes (all zero rows) can be found easily as well because row_ptr[i] == row_ptr[i+1] if the node i is dangling.

See CRS Format
