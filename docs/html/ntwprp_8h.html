Calculates the pagerank of the nodes of a web graph.

If it_specific is non 0, the parameter won't be taken into account and the algorithm will run it_specific iterations. Else if it_specific is zero, the algorithm will run until it converges to e or the number of iterations grows more than a safety limit.

Parameters
webGraphThe web graf in ntw_crs representation.
cThe teleportation coefficient.
eThe convergence's tolerance.
it_specificIf not 0, the algorithm will run it_specific iterations.
colorsThe color groups of the nodes.
streamOpened FILE* stream for printing results.
Returns
double* The calculated pagerank vector.

Calculates one iteration of \((I-A)x = b\).

Uses the Gauss-Sneidel method, using forward substitution.

\[\vec{x}_i^{k+1} = \frac{1}{1-a_{ii}}(b_i - \sum_{j=1}^{i-1} a_{ij}\vec{x}_j^{k+1} - \sum_{j=i+1}^{n} a_{ij}\vec{x}_j^k)\]

Parameters
matrixThe sparse matrix A in CRS form.
x_vecThe vector x that will be replaced.
b_vecThe vector b.
colorsThe color groups of the nodes.
